# Code base for Vue3 PWA
Using pinia, sass, typescript, vue3, working PWA already ready to go.


## Project setup
```
npm install
```
node ver: 20.15.0

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
